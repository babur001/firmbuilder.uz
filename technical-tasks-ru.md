ТЕХНИЧЕСКОЕ ЗАДАНИЕ
Проект: Разработка корпоративного сайта Firmbuilder.asia Назначение: ТЗ для разработчика и дизайнера

1. ОБЩИЕ ТРЕБОВАНИЯ
   Тип сайта: Корпоративный
   Стиль: Минимализм (чистый дизайн, светлая палитра, строгая типографика)
   Языки: Узбекский / Русский / Английский Адаптивность: Desktop / Tablet / Mobile
   CMS: WordPress или Laravel (по согласованию) SSL-сертификат: Обязательно
2. СТРУКТУРА САЙТА
   Главная страница:

- Описание компании
- Направления: Торговля / Логистика / Консалтинг - География: Китай – Россия – Азия – США
- Преимущества
- Контактная форма
  Раздел «Торговля»:
- Поиск и проверка поставщиков
- Закупка и перепродажа товаров
- Продажа техники (каталог + карточка товара)
- Завод по производству асфальта (фото, описание, мощности) - Завод по производству бетона (процесс производства, фото)

Раздел «Логистика»:

- Доставка из Китая
- Направления: Россия / Азия / США
- Формат: доставка до двери
- Морские / Авиа / Авто / ЖД перевозки - Таможенное оформление
- Консолидация грузов
- Форма расчета стоимости
  Раздел «Консультационные услуги»: - Регистрация компании в Китае
- Регистрация компании в Гонконге - Подготовка пакета документов
- Визовый сервис (бизнес, рабочая, туристическая, групповая) - Форма заявки
  Раздел «Контакты»:
- Офис в Китае (адрес, телефон, email, карта)
- Офис в Узбекистане (адрес, телефон, email, карта) - Общая форма обратной связи

3. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

- SEO-структура URL
- Meta Title / Description
- Sitemap.xml
- Robots.txt
- Google Analytics
- Защита от спама (reCAPTCHA)
- Возможность добавления новостей

- Подготовка под интеграцию CRM

4. ТРЕБОВАНИЯ К ДИЗАЙНУ

- Минимализм
- Светлый фон
- Акцентные цвета: темно-синий / графит
- Современные иконки
- Реальные фотографии (логистика, производство, офис) - Удобная навигация
- Четкая типографика
